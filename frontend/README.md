# Frontend - GPC Resume Builder

React + Vite frontend for the GPC Resume Builder platform.

## Features

- Landing page and dashboard
- Resume builder forms
- Resume preview page
- Public resumes page (preview-only)
- Authentication UI flow
- API integration with backend

## Tech Stack

- React
- Vite
- Tailwind CSS
- Redux Toolkit
- Axios

## Environment Variables (`frontend/.env`)

```env
VITE_API_URL=http://localhost:3000
```

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
